# BELPORTAL - Белорусский портал

Веб-сайт BELPORTAL - это информационный портал о Беларуси, созданный по дизайну изображения.

## Структура проекта

```
belportal/
├── index.html      # Главная страница
├── styles.css      # Стили CSS
└── README.md       # Документация
```

## Описание

Сайт включает в себя:

- **Шапка (Header)**: Логотип BELPORTAL и навигационное меню (Главная, Новости, Статьи, Контакты)
- **Hero секция**: Приветственный блок с заголовком и кнопкой призыва к действию
- **Секция новостей**: Карточки с последними новостями (3 карточки)
- **Футер**: Копирайт и информация об авторских правах

## Цветовая схема

- Основной цвет (темно-синий): `#1e3a5f`
- Акцентный цвет (бирюзовый): `#4fd1c5`
- Фон: `#f5f5f5`
- Текст: `#333`

## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- Адаптивный дизайн (Responsive Design)

## Как запустить

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер (например, через VS Code Live Server)

## Адаптивность

Сайт адаптирован для:
- Десктоп (> 768px)
- Планшет (768px - 480px)
- Мобильные устройства (< 480px)
